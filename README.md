# lighttube - A lightweight, privacy respecting alternative frontend for YouTube

---

## Features

- Lightweight
- Ad-free
- Dislike counts from [Return YouTube Dislike](https://www.returnyoutubedislike.com/)
- Subscription feed
- Proxying the videos through LightTube
